====== Docker ======

https://docs.docker.com/

**Versions**

* Docker CE : https://store.docker.com/search?offering=community&q=&type=edition moteur et orchestrateur open source
	* Docker CE Edge : une version par mois
	* Docker CE Stable : 4 versions par an
* Docker EE https://www.docker.com/enterprise-edition support, backport de patch (LTS sur 1 an), gestion des utilisateurs, scan de sécurité des images
* Moby https://mobyproject.org/ : les briques de base, en open source, et à destination des concepteurs.

**Docker, c'est aussi…**

* containerd https://containerd.io/ sous le moteur docker. Est aussi utilisé par l'orchestrateur Kubernetes. Swarm est l'orchestrateur de Docker. 
* Linux kit https://blog.docker.com/2017/04/introducing-linuxkit-container-os-toolkit/ pour builder des images minimalistes et sécurisés de Linux qui font tourner un docker.
* InfraKit https://blog.docker.com/2017/01/infrakit-hood-high-availability/ construire un cluster résilient

Dans Docker for Azure, Docker for AWS…, les outils Linux Kit, InfraKit etc sont utilisés.

**Tips**

Pour lancer les commandes docker depuis un user:
sudo usermod -aG docker <<user>>
