===== Sous Windows et Mac (Docker CE 64 bits) =====
Historiquement, il fallait utiliser Boot to Docker. Par la suite, c'est Docker machine https://docs.docker.com/machine qui a pris le relai comme un installer. Au début peu fiable, cela s'est amélioré vers l'année 2016.

Finalement, le nouveau standard, en 2017 est un installer dédié, Docker for Mac https://www.docker.com/docker-mac et Docker for Windows https://www.docker.com/docker-windows.
