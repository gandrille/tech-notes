===== Versions =====
* [[https://store.docker.com/search?offering=community&q=&type=edition|Docker CE]] : moteur et orchestrateur open source
	* Docker CE Edge : une version par mois
	* Docker CE Stable : 4 versions par an
* [[https://www.docker.com/enterprise-edition|Docker EE]] support, backport de patch (LTS sur 1 an), gestion des utilisateurs, scan de sécurité des images
* [[https://mobyproject.org/|Moby]] : les briques de base, en open source, et à destination des concepteurs.

===== Docker, c'est aussi… =====
* [[https://containerd.io/|containerd]] sous le moteur docker. Est aussi utilisé par l'orchestrateur Kubernetes. Swarm est l'orchestrateur de Docker. 
* [[https://blog.docker.com/2017/04/introducing-linuxkit-container-os-toolkit/|Linux kit]] pour builder des images minimalistes et sécurisés de Linux qui font tourner un docker.
* [[https://blog.docker.com/2017/01/infrakit-hood-high-availability/|InfraKit]] construire un cluster résilient

Dans Docker for Azure, Docker for AWS…, les outils Linux Kit, InfraKit etc sont utilisés.
