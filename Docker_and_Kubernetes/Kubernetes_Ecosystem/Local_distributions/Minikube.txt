====== Minikube ======
https://minikube.sigs.k8s.io

* Minikube is using Docker Engine
* Under macOS, uses [[https://minikube.sigs.k8s.io/docs/drivers/hyperkit/|HyperKit]], an open-source hypervisor
* no GUI

===== Commands =====
''minikube version''

**Lifecycle**
* ''minikube start''
* ''minikube status''
* ''minikube stop''

**Minikube CRI-O**
[[https://cri-o.io/|CRI-O]] is a lightweight container runtime,  optimized for Kubernetes.
To start Minikube with CRI-O as container runtime, instead of Docker: ''minikube start --container-runtime=cri-o'' 

''minikube dashboard''

''minikube ssh''
''minikube ip''
''minikube service my-service''
