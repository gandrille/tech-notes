**Knative is an Open-Source Enterprise-level solution to build Serverless and Event Driven Applications Serverless Containers in Kubernetes environments.**

* Project started by Google in 2018
* 2022 Donated by Google to CNCF in 2022

Framework for scaling containers based on requests on HTTP endpoints,...

===== Knative event source =====
https://knative.dev/docs/eventing/sources/

* Redis stream
* GitHub / GitLab
* Topic Kafka
* PigSource (cron job)
* AWS services, GCP services
