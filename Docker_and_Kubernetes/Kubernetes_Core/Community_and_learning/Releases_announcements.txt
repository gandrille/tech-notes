====== Releases announcements ======

===== 2020-08-26 Kubernetes 1.19 =====
[[https://kubernetes.io/blog/2020/08/26/kubernetes-release-1.19-accentuate-the-paw-sitive/|Release notes: Accentuate the Paw-sitive]]

**Major Themes**
* Increase Kubernetes support window to one year
* Storage capacity tracking
* Generic ephemeral volumes
* CSI Volume Health Monitoring
* Ingress graduates to General Availability
* Structured logging
* New klog methods
* Client TLS certificate rotation for kubelet

===== 2020-03-25 Kubernetes 1.18 =====
[[https://kubernetes.io/blog/2020/03/25/kubernetes-1-18-release-announcement/|Release notes: Fit & Finish]]

**Major Themes**
* Kubernetes Topology Manager Moves to Beta - Align Up!
* Serverside Apply Introduces Beta 2
* Extending Ingress with and replacing a deprecated annotation with IngressClass
* SIG-CLI introduces kubectl alpha debug
* Introducing Windows CSI support alpha for Kubernetes
