====== JSON processing (JSON-P) ======
**Main features**
* Build JSON objects (using a fluent API)
* Read JSON objects
* Write JSON objects / Stream JSON data
* Transform JSON
	* [[https://tools.ietf.org/html/rfc6901|JsonPointer]] helps to get the value, remove it, replace it, add a new and check for existence
	* [[https://tools.ietf.org/html/rfc6902|JsonPatch]] to define a set of operations to apply on a given JSON
	* [[https://tools.ietf.org/html/rfc7386|JsonMergePatch]] for merging two JSON objects
