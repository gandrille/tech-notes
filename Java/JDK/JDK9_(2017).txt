====== JDK9 (2017) ======
**Sources**
* https://howtodoinjava.com/java9/java9-new-features-enhancements/

**Key features**
* Jigsaw, le système modulaire
* Méthodes privées dans les interfaces
* Better Try-with-resources
* Instanciation de collections immuables
* Better annotation @Deprecated 
* API Flow
* JShell
* Améliorations de l’API Stream (takeWhile, dropWhile,...)
* Deprecated
   - Applets et la plateforme de déploiement Web Start
